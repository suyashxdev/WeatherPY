WeatherPY has already received an important performance upgrade with a SQLite-based caching system, and more improvements are planned for the next update.

Implemented recently:

- A local cache powered by SQLite to reduce repeated API calls and speed up repeated city lookups.
- Faster response handling for recently requested weather data.

Upcoming improvements for the next version include:

1. A more polished and interactive user interface built with Python's Textual library.
2. Better search history, saved preferences, and a more personalized experience.
3. Enhanced error handling, smoother navigation, and richer weather details.

These updates aim to make the app faster, more efficient, and even more user-friendly.
